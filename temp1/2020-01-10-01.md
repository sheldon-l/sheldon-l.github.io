- sign up a repl.it account;
- add a new repl of `Django`;
- after repl.it waking up, click start;

```bash
# `ctl + .` command bar

# `ctl + shift + s` open shell

# if manage.py permission denyed:
# `ctl + shift + p` in repl page to open shell
chmod +x manage.py
```
